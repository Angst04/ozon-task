# Progress Component

Компонент индикатора прогресса для веб-приложений

## Функционал
- Ввод числового значения прогресса в диапазоне от 0 до 100
- Переключатель режима анимации вращения
- Переключатель видимости компонента

## Структура проекта

- `index.html` - разметка приложения
- `style.css` - стили, анимации и медиа-запросы
- `script.js` - логика компонента (`ProgressAPI`) и обработка событий

## Принцип работы
### Инициализация
Метод init(elementId) принимает идентификатор DOM-элемента, находит внутри него SVG-структуру и вычисляет длину окружности для корректного отображения штриховой линии.

### Структура рендеринга
Компонент использует SVG с двумя элементами circle:

- progress-bg - статичный фоновый трек
- progress-bar - динамическая линия прогресса, управляемая через stroke-dasharray и stroke-dashoffset

### Управление состоянием
Внутреннее состояние модуля хранит текущие параметры:

- value - текущий процент заполнения (0–100)
- state - режим работы (normal, animated)
- circumference - вычисленная длина окружности
